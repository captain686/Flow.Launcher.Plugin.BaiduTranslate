# Flow Launcher 百度翻译插件

+ 🚀 需要python环境
+ 🤦‍♂️ 需要注册[百度翻译开发者](https://fanyi-api.baidu.com/)，并开通[`通用翻译`](https://fanyi-api.baidu.com/product/11)

## Use

下载项目到`FlowLauncher`安装目录的`Plugins`目录中

```bash
pip install -r requirements.txt
```

重启`FlowLauncher`
